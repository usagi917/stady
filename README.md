# 📈 株価データ表示アプリ


## 🌟 このアプリでできること

- 📊 **リアルタイム株価データ**を取得して表示
- 📈 **ローソク足チャート**で株価の動きを視覚化
- 📉 **移動平均線**で株価のトレンドをチェック
- 📋 **出来高チャート**で取引量も確認
- 💾 **CSVファイル**でデータをダウンロード
- 🇯🇵 **日本株**と🇺🇸 **米国株**の両方に対応

## 🚀 始め方

### 1. 必要なものをインストール

まずはPythonが入ってるか確認！
入ってなかったら[Python公式サイト](https://www.python.org/)からダウンロード

```bash
# ターミナル（コマンドプロンプト）を開いて、以下を実行
pip install streamlit yfinance pandas plotly
```

### 2. アプリを起動

```bash
# このフォルダに移動
cd /Users/あなたのフォルダに移動

# アプリを起動！
streamlit run main.py
```

### 3. ブラウザで確認

自動でブラウザが開くので確認

## 📱 使い方

### 基本的な操作

1. **サイドバー**（左側）で銘柄コードを入力
2. **期間**を選択（開始日と終了日）
3. **株価チャート**と**データ**が自動で表示される
4. 必要に応じて**CSVダウンロード**

### 銘柄コードの入力方法

#### 🇯🇵 日本株の場合
- 証券コード + `.T` を入力
- 例：
  - `7203.T` → トヨタ自動車
  - `6758.T` → ソニーグループ
  - `9984.T` → ソフトバンクグループ
  - `8306.T` → 三菱UFJフィナンシャル・グループ

#### 🇺🇸 米国株の場合
- ティッカーシンボルをそのまま入力
- 例：
  - `AAPL` → Apple
  - `GOOGL` → Google (Alphabet)
  - `TSLA` → Tesla
  - `MSFT` → Microsoft
  - `AMZN` → Amazon

## 🎯 画面の見方

### メイン画面
- **💰 最新価格**: 最新の株価
- **📈 変動率**: 前日比の変動率
- **📊 最高値**: 期間中の最高値
- **📉 最安値**: 期間中の最安値

### チャート
- **ローソク足チャート**: 株価の動きを詳しく表示
  - 🟢 緑色 = 上昇（終値 > 始値）
  - 🔴 赤色 = 下落（終値 < 始値）
- **出来高チャート**: 取引量を表示
- **移動平均線**: 株価のトレンドを表示
  - オレンジ線 = 5日移動平均
  - 赤線 = 25日移動平均

## 🛠️ トラブルシューティング

### よくあるエラーと解決方法

#### 「データが見つかりませんでした」
- 銘柄コードが間違ってる可能性があるよ
- 日本株の場合は `.T` を忘れずに！
- 土日祝日は市場が閉まってるから最新データが取れないかも

#### 「モジュールが見つかりません」
```bash
# 必要なライブラリを再インストール
pip install --upgrade streamlit yfinance pandas plotly
```

#### アプリが起動しない
```bash
# Streamlitのバージョンを確認
streamlit --version

# 最新版にアップデート
pip install --upgrade streamlit
```

## 📚 使用しているライブラリ

## 💡 カスタマイズのヒント

### デフォルト銘柄を変更したい
`main.py` の32行目を編集してね：
```python
default_ticker = '7203.T'  # ここを好きな銘柄に変更
```

### 期間を変更したい
`main.py` の48行目を編集してね：
```python
default_start_date = today - timedelta(days=365)  # 365日前から → 好きな日数に変更
```
